Week 1: Getting Started

Requirements to Begin:

  • Install Python
  • Install Sublime
  • Setup your github student Development Pack
  • Learn git syntax
  • Learn Python Syntax at Codecademy
  • Learn Unix Syntax at Codecademy
  • Read the additional resources to learn more about Python and Git

Lets build our first program!

  1. Open up Sublime
  2. Create a new document(File->New)
  3. Copy this line of codeprint("Congrats, your program works!!") into your document
  4. Save the document to a perfered location with the extension ".py" such as test2.py
  5. Open up Terminal (OSX) or cmd.exe (Windows) and using Unix commands navigate to where you saved your document.
  6. Once in the directory/location of where you sved your file we can now run your program. type "python" and then the name of your program. For example python test2.py and then click enter.
  7. If you see Congrats, you program works!! printed in your terminal/cmd screen you have completed your first program!

Challenge 1

Write a function that takes a character (i.e. a string of length 1) and returns True if is a vowel, False otherwise.

Challenge 2

  • Write a program to add all the numbers that divisible by 3 (OR) 5 below 10000. For example if we want sum of numbers below 15, we add 3+5+6+9+10+12 = 45
  • Write a program to add all the numbers that divisible by 3 (AND) 5 below 10000. For example if we want sum of numbers below 50, we add 15+30+45 = 90

Challenge 1

Define a function is_palindrome() that recognizes palindromes (i.e. words that look the same written backwards). For example, is_palindrome("radar") should return True.

Challenge 2

Write a Python program to get the Fibonacci series between 0 to 50.
Note : The Fibonacci Sequence is the series of numbers :0, 1, 1, 2, 3, 5, 8, 13, 21, ....
Every next number is found by adding up the two numbers before it.
Expected Output : 1 1 2 3 5 8 13 21 34
